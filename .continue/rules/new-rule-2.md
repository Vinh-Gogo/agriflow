Create a `HeroSection` component for the homepage.

**Functional Requirements:**
- Large impressive headline (H1).
- Primary and Secondary CTA (Call to Action) buttons.
- A short description paragraph.

**Vibe & Design Requirements:**
- **Background:** Pure black.
- **Effect:** Behind the headline text, there should be a subtle "glow" effect in pink/purple (using CSS blur or absolute divs with gradient).
- **Typography:** Sans-serif font, modern, bold.
- **Animation:** Text should fade-in from bottom to top when the page loads.
- **Responsive:** Must look good on mobile and desktop.

Use Tailwind CSS for all styling. Code must be clean, separating logic if necessary.
