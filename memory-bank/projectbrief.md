# Project Brief: HydroSense Hub

## Core Requirements
- **Real-time Monitoring**: Track soil moisture, temperature, humidity, and water pressure across multiple irrigation zones.
- **AI-Powered Recommendations**: Utilize Genkit and Gemini to calculate a "Thirst Index" for each zone and provide natural language explanations for watering needs.
- **Automated Scheduling**: Manage recurring irrigation routines tailored to different plant types (Fruit trees, Vegetables, Lawn).
- **Manual Control**: Provide an override system to start/stop watering manually for specific zones.
- **Analytics & Reporting**: Generate weekly performance summaries using AI to optimize water usage.

## Goals
- Empower users with data-driven gardening.
- Maximize water efficiency through environmental sensor integration.
- Provide a professional, responsive, and intuitive UI for both mobile and desktop users.
